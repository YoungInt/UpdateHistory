## 해피 네임 스티커 미니 3.0.0 릴리즈
# 변경사항
 - 관리자 모드 진입 방법 변경
   - '나', '을', '해', '요', 벌(그림) 을 순서대로 터치.

 - 업데이트 방식 변경   
   - 관리자 모드의 관리자 메뉴(고급) 탭 하단에 업데이트 버튼 추가.
   - 모티브 업데이트, 동영상 업데이트, 프로그램 업데이트 버튼 추가.
   - 프로그램 시작 시 최신 버전 체크, 업데이트 체크 실패시 3분 간격으로 3회 재시도.
   - 업데이트 가능 일 때는 업데이트 가능한 최신버전 정보 버튼에 출력.
   - 모티브 업데이트는 프로그램 시작시 자동으로 진행.
   - 동영상 업데이트는 수동으로 버튼 클릭 후 프로그램 다운로드 받은 뒤 자동 적용.
   - 프로그램 업데이트는 수동으로 버튼 클릭 후 프로그램 다운 받은 뒤 현재 실행중인 프로그램 종료 후 설치 화면으로 전환, 수동으로 설치 진행.

 - 업데이트 표기 방식 변경
   - "2017112901"과 같은 표기에서 "30"으로 변경
   - 앞자리는 메이저, 뒷자리는 마이너 버전으로 보통 회의에서 나오는 이슈들의 변경으로 인한 업데이트 등은 마이너 버전 증가
   - 현재 이 방법도 임시적인 방법으로 차후에 "3.0.0"과 같은 방식으로 변경 예정

 - 업데이트 히스토리 추가
   - 설치 프로그램과 함께 해당 버전의 업데이트 변경 정보 파일 추가 파일 이름은 "Release 3.x.x"의 형태.

 - 다이소 버전 추가
   - 전용 프로그램 설정에 다이소 버전 추가.
   - 디폴트는 해피 버전이니 다이소에 설치 후 다이소 버전으로 변경 필요.

- 카드 단말기 테스트 모드 추가
  - 관리자 모드의 관리자 메뉴(고급) 탭에 단말기 테스트 모드 추가.
  - 카드 단말기 테스트 모드 활성화 시 결제 금액이 10원으로 변경. (주의 요망)
  - 실수 방지하기 위해 프로그램 재실행 시 다시 비활성화.
