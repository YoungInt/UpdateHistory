# 해피 네임 스티커 미니 3.2.0 릴리즈

## 변경사항

 - 관리자 모드에 한장 출력 추가(한 조각 출력)

 - 용지 최대 출력값 Default값 3자리에서 4자리로 변경

 - 대형 프린터(TSC-345)와 소형 프린터(TA310)의 선택 가능
   - 관리자 모드의 라벨프린터 탭에서 변경 가능
   - 대형과 소형에서 모두 소형 용지 사용 가능, *(대형 용지 사용 불가)
   - 해당 기계에 맞는 프린터 선택인지 확인(테스트 출력) 필수

 - 출력되는 스티커 레이아웃 변경
   - 기존의 출력문구 위치값 초기화후 재 조정 필요

 - 프로그램 아이디 선택 방식 변경 (라디오버튼 -> 드롭 다운)

 - 관리자 모드 타이틀의 Mini -> Standard로 변경

 - 설치 시 바로가기 이름에서 미니 제거

 - 인스톨러 실행시 설명문에서 Mini -> Standard로 변경

 - 모니터링 서버에서 프로그램의 프린터 설정값에 따라 "대형", "소형"으로 표기






# 해피 네임 스티커 미니 3.1.2 릴리즈

# 해피 네임 스티커 미니 3.1.1 릴리즈

## 변경사항

- 관리자 모드의 프로그램 나가기 팝업에 버튼 추가
  - 저장 후 프로그램 종료 : 프로그램의 config 파일을 백업 후 프로그램을 종료
  - 되돌아가기 : 관리자 모드로 다시 복귀

- 업데이트 새로고침 버튼 추가
  - 관리자 모드에서 업데이트 상태를 새로고침 할 수 있는 버튼 추가

- 비밀번호 입력의 관리자 모드 진입 방식 적용
  - 모티브 선택 페이지, 일시 품절 페이지에 적용
  - 5초안에 아무런 입력이 없으면 팝업 종료, 누르면 다시 제한 시간 초기화
  - 기존의 진입 방법과 같이 사용 가능

- 최대 출력 가능 장수가 1장일 때 출력 장수 선택 UI의 '+', '-' 버튼 비활성화

- 구몬 버전의 진입 방법 변경
  - 기존의 진입 방식 제거, 비밀번호 진입 방식으로 변경

- 업데이트 버전표기 변경
  - 프로그램 내 표기가 기존 "31"에서 "311"과 같은 방식으로 변경 (기존의 "3.1.1"과 같은 방식의 두자리에서 세자리 표기로 변경)

- 네트워크 상태 확인 방법 변경
  - Ping 추가

- 라우터가 모두 소진되었을 때 카드 결제가 비활성화 되지 않던 현상 수정

- 기존의 프로그램 시작 시 실행되는 AutoUpdate 프로그램 제거 (예전 업데이트 프로그램)

# 해피 네임 스티커 미니 3.0.0 릴리즈

## 변경사항

 - 관리자 모드 진입 방법 변경
   - 내부적으로 공지

 - 업데이트 방식 변경   
   - 관리자 모드의 관리자 메뉴(고급) 탭 하단에 업데이트 버튼 추가.
   - 모티브 업데이트, 동영상 업데이트, 프로그램 업데이트 버튼 추가.
   - 프로그램 시작 시 최신 버전 체크, 업데이트 체크 실패시 3분 간격으로 3회 재시도.
   - 업데이트 가능 일 때는 업데이트 가능한 최신버전 정보 버튼에 출력.
   - 모티브 업데이트는 프로그램 시작시 자동으로 진행.
   - 동영상 업데이트는 수동으로 버튼 클릭 후 프로그램 다운로드 받은 뒤 자동 적용.
   - 프로그램 업데이트는 수동으로 버튼 클릭 후 프로그램 다운 받은 뒤 현재 실행중인 프로그램 종료 후 설치 화면으로 전환, 수동으로 설치 진행.

 - 업데이트 표기 방식 변경
   - "2017112901"과 같은 표기에서 "30"으로 변경
   - 앞자리는 메이저, 뒷자리는 마이너 버전으로 보통 회의에서 나오는 이슈들의 변경으로 인한 업데이트 등은 마이너 버전 증가
   - 현재 이 방법도 임시적인 방법으로 차후에 "3.0.0"과 같은 방식으로 변경 예정

 - 업데이트 히스토리 추가
   - 설치 프로그램과 함께 해당 버전의 업데이트 변경 정보 파일 추가 파일 이름은 "Release 3.x.x"의 형태.

 - 다이소 버전 추가
   - 전용 프로그램 설정에 다이소 버전 추가.
   - 디폴트는 해피 버전이니 다이소에 설치 후 다이소 버전으로 변경 필요.

- 카드 단말기 테스트 모드 추가
  - 관리자 모드의 관리자 메뉴(고급) 탭에 단말기 테스트 모드 추가.
  - 카드 단말기 테스트 모드 활성화 시 결제 금액이 10원으로 변경. (주의 요망)
  - 실수 방지하기 위해 프로그램 재실행 시 다시 비활성화.
